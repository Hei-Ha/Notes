SVG 和 Canvas的区别

SVG 和 Canvas 的区别

区别

  1. Canvas 主要是用来绘制 2D 图形的。
  2. SVG 主要是用标签来绘制不规则矢量图的。
  3. SVG 渲染节点过多的时候渲染慢，Canvas 性能更好一点，但是写起来比较麻烦。
